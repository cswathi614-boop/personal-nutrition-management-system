# Personal Nutrition Management System

## Project Description
The Personal Nutrition Management System is a frontend web application designed to provide diet suggestions based on different health conditions. The project focuses on creating a simple, user-friendly interface using basic web technologies.

## Features
- Patient details form
- Health condition based diet suggestions
- Simple and responsive UI
- Easy navigation between pages

## Technologies Used
- HTML
- CSS
- JavaScript

## Pages
- Home Page
- Patient Details Page
- Diet Suggestion Page

## Purpose
This project was originally developed as part of my academic work and later recreated using modern web technologies to improve my frontend and UI development skills.
